Descriptions and parameter settings

CBioExplorer allows the users to internally validate and calibrate the nomogram based on bootstrap and calibration analysis. Figure 23 shows internal validation for the prediction model in the training set and test set.

  • Prediction years: Define the time points in years you want to predict based on the nomogram. The longest time point should not exceed the max of survival (relapse) duration.

  • Bootstrap iterations: Number of bootstrap resamplings.

  • Ratio: Ratio of resamplings.
